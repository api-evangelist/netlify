title: "Environment variable size limit removed for Serverless Functions"

date: "2026-06-12"

You can now use as many environment variables as your Serverless Functions need. The 4KB total size limit on environment variables no longer applies to functions running on the current Netlify Functions runtime. This limit was a common source of friction.
