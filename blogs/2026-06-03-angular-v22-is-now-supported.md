Netlify announced day-one support for Angular v22, released on June 3, 2026. Developers can upgrade by updating @netlify/angular-runtime to v4.0.0 or later and ensuring Node.js 22.12.0 or later is in use. Key changes include automatic handling of the new allowedHosts configuration to prevent HTTP 400 errors on preview deployments and built-in support for forwarded headers through the trustProxyHeaders option.
